# Utils
开发中常用到的一些工具类

# 用法

在根目录下的 build.gradle 文件中添加

```groovy
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

然后在app目录下的build.gradle文件中添加

```groovy
	dependencies {
	        compile 'com.github.andyNinty:CustomUtils:1.0'
	}
```
